= Atlrug-website

This is a Ruby-on-Rails web application for the ATLRUG (Atlanta Ruby User Group).

* **master:** [![Build Status master](https://secure.travis-ci.org/jasnow/atlrug-website.png?branch=master)](http://travis-ci.org/jasnow/atlrug-website)

* [![Dependency Status](https://gemnasium.com/jasnow/atlrug-website.png?travis)](https://gemnasium.com/atlrug-website)


== Setup

1. To install gems: <tt>bundle install</tt>.

On 4/18/2013, the requirements were: Ruby 1.9.3; Rails 3.2.13.

2. Create databases and tables:

    rake db:setup
    rake db:migrate

3. Run the tests by typing "rake".

4. Project is on Github at https://github.com/jasnow/atlrug-website.

== To Contribute

1. Fork project on Github.

2. Clone it locally.

3. Make Change.

4. Push it back to Github.

5. Create a Pull Request to original repo.
